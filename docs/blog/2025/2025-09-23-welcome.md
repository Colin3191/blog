## 关于这个博客

这个博客使用 Rspress 构建，是一个基于 Rust 的静态站点生成器。我选择它的原因包括：

- **快速构建**：基于 Rust 的构建工具链，提供极快的编译速度
- **MDX 支持**：可以在 Markdown 中使用 React 组件
- **全文搜索**：内置全文搜索功能
- **现代化设计**：支持暗色主题，响应式设计
